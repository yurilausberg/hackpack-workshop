# HackPack — Agents Hackathon Workshop

Welcome! This repository hosts the **HackPack** resources for the **Agents Hackathon** workshop. Everything you need to get started is right here.

## 📦 What's in the HackPack

The HackPack is a PowerPoint deck that walks you through the hackathon: goals, agenda, use cases, tooling, and step-by-step guidance for building your AI agents.

- **[⬇️ Download HackPack.pptx](https://github.com/yurilausberg/hackpack-workshop/raw/main/resources/HackPack.pptx)** — the full workshop deck (~32 MB). Clicking this downloads the file directly.

## ⬇️ How to get the HackPack

Pick whichever is easiest for you:

1. **One-click download (recommended)**
   - Click **[⬇️ Download HackPack.pptx](https://github.com/yurilausberg/hackpack-workshop/raw/main/resources/HackPack.pptx)** — the file downloads immediately, no extra pages
   - Open the downloaded file in PowerPoint (desktop or web)

2. **Download the whole repo as a ZIP**
   - Click the green **`< > Code`** button at the top of the repo
   - Choose **Download ZIP**, then unzip and open `resources/HackPack.pptx`

3. **Clone with Git** (for the command-line inclined)
   ```bash
   git clone https://github.com/yurilausberg/hackpack-workshop.git
   cd hackpack-workshop
   ```

## 🚀 Getting Started

1. Download and open **`HackPack.pptx`**.
2. Work through the deck in order — it's designed to take you from setup to a working agent.
3. Follow along with the facilitators during the live sessions.
4. Bring your questions and ideas — the best use cases come from the room!

## 💡 Tips

- View the deck in **Slide Show mode** for any embedded animations or links.
- Keep a copy handy offline; Wi-Fi at events can be unpredictable.
- Have PowerPoint (desktop or [PowerPoint for the web](https://office.com)) ready before the session starts.
